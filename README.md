# GodoFredo - Blog - Ghost
### 说明

原文件来自@GodoFredoNinja

## 使用

### 全局安装
```
$ npm install -g stylus
$ npm install -g gulp
$ npm install -g bower
```

### 本地安装
$ npm install
$ bower install
```
### 开发模式
```
$ gulp bower
$ gulp
```

#### 编辑页面底部 `parttials/footer.hbs`

#### 编辑社交按钮 `post.hbs`

#### 编辑右栏 `partials/sidebar.hbs`
